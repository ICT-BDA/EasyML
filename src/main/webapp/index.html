STUDIO

Easy Machine Learning

A large-scale data analysis platform developed by the Big Data Analysis Group of the Chinese Academy of Sciences (CAS), which is designed to help users reduce the threshold of large data analysis and solve the problem of transforming from raw data to analysis results. It includes two major components: distributed large data analysis function and algorithm library EasyML Lib and visual task construction and management platform EasyML Studio.These can greatly enhance the user's analysis and efficiency of large data analysis.

EML Lib

  1. Spark Distributed Computing Framework
  2. Extensive ML algorithms are available
  3. Standalone/data and model distributed
  4. API interface/support command line running

EML Studio

  1. Drag-and-drop graphical user interface
  2. Standalone and Distribute mixed execution
  3. Integrated rich system analysis program
  4. Private/custom data and algorithm module
  5. Program modules/application sharing
  6. Large data analysis sample templates

Easy to use
Provides a draggable graphical interface to help users quickly build analysis tasks.

Large scale scalable
The internal algorithm is based on the Spark memory distributed computing framework, with powerful large data processing capabilities.

In-depth analysis 
With a rich machine learning algorithm, covering classification clustering, text analysis, personalized recommendation and so on.

Mixed calculation
Support Mapreduce, Spark and stand-alone parallel execution of the program, stand-alone / distributed algorithm.

Open and sharing
Support the release and sharing of data, program modules and analysis tasks, and reduce the time cost of the user.
